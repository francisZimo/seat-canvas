# TC-movable-area-view


可移动的视图容器，在页面中可以拖拽滑动或双指缩放。

movable-view必须在movable-area组件中，并且必须是直接子节点，否则不能移动。



##引入组件即可

```
<template>
	<view>
		<TC-movable-area-view></TC-movable-area-view>
	</view>
</template>

```
